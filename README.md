📊 Análise de Vendas com SQL e Power BI

Projeto de análise de vendas desenvolvido utilizando SQL, modelagem dimensional e Power BI, com foco em organização de dados, tratamento ETL e criação de indicadores de negócio.

🚀 Tecnologias Utilizadas
SQL
Oracle Database 21c XE
SQL Developer
Power BI
Excel
Modelagem Dimensional
📌 Objetivo

Transformar múltiplas planilhas de vendas em uma estrutura organizada para análise de dados e geração de dashboards estratégicos.

🏗️ Estrutura do Projeto
Tabelas Dimensão
Cliente
Loja
Tempo
Tabelas Fato
Venda_Produtos
Devoluções
🛠️ Etapas Desenvolvidas
Levantamento de necessidades de negócio
Criação do modelo dimensional
Tratamento ETL dos dados
Validação de integridade
Consultas SQL analíticas
Integração Oracle + Power BI
Desenvolvimento de dashboards
📊 Indicadores Criados
Total vendido por loja
Taxa de devolução
Perfil de clientes
Média de vendas por colaborador
Vendas por período
Distribuição geográfica de vendas
